# 我的 Python 小游戏合集 🎮

这里是我写的一些简单 Python 小游戏，适合新手学习和娱乐！

## 包含的游戏
1.  **猜数字 (guess_number.py)**
    *   规则：电脑随机生成一个数字，你来猜，它会提示你猜大了还是猜小了，直到猜对为止。
    *   运行：`python guess_number.py`

2.  **小猪快跑 (pig_emoji.py)**
    *   规则：一只可爱的小猪 🐷 在横线上向前奔跑，跑到终点后会显示恭喜信息。
    *   运行：`python pig_emoji.py`

## 怎么运行？
1.  确保你电脑安装了 Python 3.x。
2.  下载或克隆这个仓库到本地。
3.  打开命令行，进入项目文件夹。
4.  输入 `python 文件名.py` 运行对应的游戏。
